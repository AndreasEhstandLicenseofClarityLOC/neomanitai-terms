# Security Policy — NEOMANITAI Compendium

## Reporting Security Issues

If you discover a security vulnerability or an issue with this repository, please report it via email: **augmanitai [at] proton [dot] me**

Do not open a public GitHub issue for security concerns.

## Scope

This security policy covers:
- The NEOMANITAI Compendium website and all term pages
- The GitHub repository and its contents
- JSON-LD metadata and structured data

## Acceptable Use

This lexicon is a descriptive research resource. The following uses are **prohibited**:
- Using terminology to develop manipulation, persuasion, or deception techniques
- Using content for AI/ML training without written permission (§17)
- Commercial exploitation (CC BY-NC-ND 4.0)
- Automated scraping or mass downloading
- Creating derivative works
- Archiving without permission

## Content Classification

- **Age restriction:** 18+ (Adults only)
- **Content type:** Descriptive research terminology
- **License:** CC BY-NC-ND 4.0 International
- **Jurisdiction:** German law, Munich
